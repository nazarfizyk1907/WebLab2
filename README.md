# QuickLink — URL Shortener (Vue.js + Node.js)

**QuickLink** — це повноцінний вебдодаток клієнт-серверної архітектури для скорочення URL-адрес. У цій версії (ЛР №3) інтерфейс був повністю переписаний на фреймворк **Vue.js**, а дані завантажуються динамічно з сервера.

## 🛠 Технологічний стек

* **Frontend**: Vue.js 3 (Options API), HTML5, CSS3 (Bootstrap 5).
* **API Client**: Axios (для асинхронних запитів).
* **Backend**: Node.js, Express.js.
* **Database**: SQLite.
* **Storage**: localStorage (для збереження сесії користувача).

## 🚀 Основний функціонал

* **Аутентифікація**: реєстрація та вхід із ідентифікацією користувача через localStorage.
* **Генерація посилань**: створення унікальних коротких кодів для довгих URL.
* **Маршрутизація (Редірект)**: автоматичне перенаправлення за коротким посиланням із підрахунком кількості переходів у реальному часі.
* **Управління даними**: динамічне додавання та видалення посилань без перезавантаження сторінки.
* **Особистий кабінет**:
    * Редагування персональних даних.
    * Відображення статистики: загальна кількість посилань та сумарна кількість переходів.

## 📂 Структура Vue-компонентів

Проєкт розділений на логічні частини (сторінки), кожна з яких працює як окремий Vue-екземпляр:

1.  **Dashboard (`index.html`)**: головна панель для управління посиланнями. Виконує GET, POST та DELETE запити.
2.  **Auth (`login.html` / `register.html`)**: обробка входу та створення акаунтів, валідація даних від сервера.
3.  **Profile (`profile.html`)**: завантаження агрегованої статистики та PUT-запити для оновлення профілю.

## 🔧 Запуск

Для роботи додатка необхідно:
1. Запустити серверну частину на Node.js (порт 3000 за замовчуванням).
2. Відкрити `index.html` або будь-яку іншу сторінку у браузері.
